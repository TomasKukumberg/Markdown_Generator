## Run project locally
### 1. Clone the repo
```bash
git clone https://github.com/TomasKukumberg/Markdown_Generator.git
```
### 2. Change into directory
```bash
cd Markdown_Generator
```
### 3. Install dependencies
```bash
npm install
```
### 4. Run the project locally
```bash
npm run dev
```
